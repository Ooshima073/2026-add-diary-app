# 日記アプリ仕様書（spec.md）

## ■ プロジェクト概要

スマートフォン上で動作する日記アプリを開発する。
ユーザーは日記を作成・閲覧・編集・検索できる。

---

## ■ 目的

- 日々の記録を簡単に残せるようにする
- React NativeとAI開発の理解を深める

---

## ■ 機能一覧

### 1. 日記一覧画面

- 日記のリストを表示する
- 無限スクロール対応

### 2. 日記作成画面

- 新しい日記を作成できる

### 3. 日記詳細画面

- 日記の詳細を表示

### 4. 日記編集画面

- 日記の内容を編集できる

### 5. 日記検索機能

- キーワードで検索できる

---

## ■ データ構造

日記データは以下を持つ：

- タイトル（必須）
- 本文（必須）
- 日付（必須）
- 画像（任意）
- アイコン（必須）

---

## ■ 使用技術

- React Native
- Expo
- JavaScript / TypeScript
- （将来的に）Firebase or Supabase

---

## ■ 開発方針

- 小さい機能から順番に作る
- AIを使って効率的に開発する
- 動くものを優先して作る
